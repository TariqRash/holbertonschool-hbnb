### Architecture Overview

The application follows a **three-layer architecture**:

1. **Presentation Layer** - API services and user interfaces
2. **Business Logic Layer** - Core models and business rules
3. **Persistence Layer** - Database operations and data access

Communication between layers is facilitated through the **Facade Pattern**.
